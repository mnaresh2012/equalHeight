##equalHeight Plugin
An useful plug-in for the e-commerce web application.
=====================
###Usage and Example -
To see an example take a look at the .
Full API reference documentation can be found from this link.
To initialze the plugin - 
```
$('#main ul li').equalHeight();
```

```
CSS - 
li {
	line-height: 30px;
}

li span {
	background: gray;
	padding: 2px 5px;
	margin-left: 15px;
	color: white;
	font-weight: bold;
}
```
